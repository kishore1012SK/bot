Chatbot Widget Sandbox

This is a mock company webpage demonstrating the plug-and-play chatbot widget. You can see the floating blue bubble in the bottom-right corner of the page.

How to embed this widget:

To embed this private, RAG-enabled chatbot into any existing company webpage, simply paste the following script tag right before the closing </body> tag: 

<script src="http://localhost:8000/static/widget.js"></script>
