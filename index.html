Integration
Two lines to go live

Add this snippet before the closing </body> tag of any webpage:

<!-- Company AI Chatbot Widget -->
<script src="https://your-chatbot-server.company.local/static/widget.js"></script>

Or use the typed React component in your SPA:

// Import the component
import ChatWidget from './components/ChatWidget'

// Render anywhere in your app
<ChatWidget
  apiUrl="https://your-chatbot-server/api/v1/widget/stream"
  title="HR Help Desk"
  themeColor="#2b5aff"
/>
